# 项目使用检查清单

使用这个清单确保你正确设置和使用了这个后端模板。

## ✅ 初始设置

### 1. 环境准备
- [ ] 已安装 Go 1.24 或更高版本
- [ ] 已安装 Make（可选但推荐）
- [ ] 已安装 Git

### 2. 项目初始化
- [ ] 克隆或复制项目到本地
- [ ] 运行 `go mod download` 下载依赖
- [ ] 复制 `.env.example` 到 `.env`
- [ ] 根据需要修改 `.env` 配置

### 3. 数据库设置
- [ ] 选择数据库类型（SQLite 或 PostgreSQL）
- [ ] 如果使用 PostgreSQL，确保数据库已创建
- [ ] 确认 `DB_DSN` 配置正确

### 4. 创建第一个用户
- [ ] 运行 `make usermgr` 构建用户管理工具
- [ ] 运行 `./bin/usermgr create` 创建用户
- [ ] 记录用户邮箱和密码

### 5. 启动服务器
- [ ] 运行 `make run` 或 `go run ./cmd/server`
- [ ] 确认服务器在 `http://localhost:8080` 启动
- [ ] 访问 `http://localhost:8080/healthz` 检查健康状态

## 📝 常见问题检查

### 问题：无法启动服务器
- [ ] 检查端口是否被占用
- [ ] 检查数据库连接
- [ ] 查看日志输出

### 问题：无法登录
- [ ] 确认用户已创建
- [ ] 检查密码是否正确
- [ ] 查看服务器日志

### 问题：API 返回 401
- [ ] 检查 JWT token 是否有效
- [ ] 确认 Authorization header 格式正确
- [ ] 检查 token 是否过期

### 问题：数据库错误
- [ ] 检查数据库连接字符串
- [ ] 确认数据库已创建
- [ ] 查看数据库日志
